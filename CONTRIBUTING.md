# Contributing

Thanks for your interest in contributing to `aicommit`.

## How to contribute

- Open an issue for bugs, ideas, or improvements
- Fork the repository
- Create a feature branch
- Make your changes
- Open a pull request

## Development setup

```bash
git clone https://github.com/andihalberkamp/aiCommit.git
cd aicommit
npm install
npm link
```

## Guidelines

- Keep the CLI simple
- Prefer readable code over clever code
- Add helpful error messages
- Update the README when behavior changes
- Add or update examples when needed

## Pull requests

Please include:

- a short summary of the change
- why the change is useful
- any manual testing notes
